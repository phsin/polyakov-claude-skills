# Конфигурация токена Scrape.do

## Быстрая настройка

1. **Скопируйте пример файла:**
   ```bash
   cp token.txt.example token.txt
   ```

2. **Вставьте ваш токен:**
   ```bash
   echo 'ваш_токен_scrape.do' > token.txt
   ```

## Где взять токен?

1. Зарегистрируйтесь на https://scrape.do
2. В личном кабинете получите API токен
3. Вставьте токен в файл `token.txt`

## Альтернативные способы

### Переменная окружения
```bash
export SCRAPEDO_TOKEN='ваш_токен'
```

### Файл .env в корне skill'а
```bash
echo 'SCRAPEDO_TOKEN=ваш_токен' > ../.env
```

## Приоритет источников токена

1. `config/token.txt` (рекомендуется)
2. Переменная окружения `SCRAPEDO_TOKEN`
3. Файл `.env` в корне skill'а

## Безопасность

- **НЕ** коммитьте файл `token.txt` в git
- **НЕ** делитесь вашим токеном
- Добавьте `token.txt` в `.gitignore`

## Проверка

После установки токена проверьте конфигурацию:
```bash
python ../scripts/check_status.py
```
